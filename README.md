# Exploratory Data Analysis (EDA) Project

## Project Overview

This project was completed as part of the DecodeLabs Data Analytics Internship Program. The objective of this project is to perform Exploratory Data Analysis (EDA) on a dataset to discover meaningful insights, identify patterns, detect outliers, and understand data distributions using Python.

## Objectives

* Load and inspect the dataset
* Perform data cleaning and preprocessing
* Calculate descriptive statistics
* Identify missing values
* Analyze trends and relationships
* Detect outliers
* Create visualizations for better understanding
* Summarize key findings

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* OpenPyXL
